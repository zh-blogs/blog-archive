严正声明：本计划的存档依赖的是互联网档案馆的存档服务，理论上存在不稳定性。另外，本计划不会为违反美利坚合众国、中华人民共和国法律的内容（有时会向中华人民共和国的相关机关报告违法内容）存档。互联网档案馆与本计划不存在任何利益联系。本计划不会收取服务费用。
